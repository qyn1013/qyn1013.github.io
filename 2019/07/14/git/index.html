常用命令

1.创建版本库

mkdir ning
cd ning
pwd //查看当前路径

2.把这个目录变成Git可以管理的仓库

git init //目录.git用来跟踪管理版本库，如果没看到则默认是隐藏的，用ls -ah可以看到

3.把文件添加到版本库

git add ning.txt
git commit -m “message” //提交文件到仓库，-m后输入本次提交的说明，可以多次add不同文件，commit可以一次提交多个文件

4.查看修改结果

git status //可以随时掌握工作区的状态
git diff //查看修改内容

5.撤销修改

git reset //回退版本

6.分支

git checkout -b //创建+切换分支
git branch //查看分支
git checkout master //分支做完，切换回master
git merge //将分支的内容融合到master上，检测有没有冲突，然后在mater分支创建一个记录合并结果的新节点
git rebase //将分支的内容apply 到或者应用到或者说将分支的头指针重定向到当前 HEAD 位置
git cherry-pick //将一个分支的内容遴选到另一个分支（类似于复制）

7.把本地分支提交到远程库

git push origin master //master是主分支

8.把最新的提交拿下来

git pull
