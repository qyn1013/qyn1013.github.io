安装Babel(能把原始代码用的箭头函数转为普通函数)

$ npm install –save-dev @babel/core

1.声明变量

1.let 声明的变量只在let所在的代码块内有效;声明的变量一定要在声明后使用；所声明的变量不再受外部影响；不允许相同作用域内重复声明同一个变量

//重新声明参数
function func(arg) {
let arg;
}
func() // 报错
function func(arg) {
{
let arg;
}
}
func() // 不报错

2.var 声明的变量在全局范围内有效

3.const 声明只读的常量，声明的常量的值不能改变（变量指向的内存地址所保存的数据不得改动）。一旦声明变量就必须立即初始化。声明后用，不可重复声明。

2.变量的解构赋值
模式匹配赋值（解构失败，变量值为undefined）

不完全解构

let [a, [b], d] = [1, [2, 3], 4];
a // 1
b // 2
d // 4

等号的右边要是数组（可遍历的结构）

指定默认值时，只有当一个数组成员等于undefined，默认值才生效

let [x = 1] = [undefined];
x // 1
let [x = 1] = [null];
x // null

变量声明、函数参数、赋值语句的模式不能带有圆括号

3.字符串

String.fromCodePoint()方法，可以识别大于0xFFFF的字符，如果String.fromCodePoint方法有多个参数，则它们会被合并成一个字符串返回。fromCodePoint方法定义在String对象上

includes()：返回布尔值，表示是否找到了参数字符串。
startsWith()：返回布尔值，表示参数字符串是否在原字符串的头部。
endsWith()：返回布尔值，表示参数字符串是否在原字符串的尾部。

repeat方法返回一个新字符串，表示将原字符串重复n次。参数为小数则取整

trimStart()消除字符串头部的空格
trimEnd()消除尾部的空格。

4.数值

Math.trunc方法用于去除一个数的小数部分，返回整数部分

Math.sign方法用来判断一个数到底是正数、负数、还是零

5.函数

指定了默认值以后，函数的length属性，将返回没有指定默认值的参数个数

6.数组

copyWithin()方法，在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原有成员），然后返回当前数组。

Array.prototype.copyWithin(target, start = 0, end = this.length) //如果start、end为负值，表示从末尾开始计算。target为负值表示倒数

find方法，用于找出第一个符合条件的数组成员
findIndex方法返回第一个符合条件的数组成员的位置，如果所有成员都不符合条件，则返回-1

Array.prototype.includes方法返回一个布尔值，表示某个数组是否包含给定的值.
indexOf方法的含义是找到参数值的第一个出现位置，要去比较是否不等于-1，内部使用严格相等运算符（===）进行判断，会导致对NaN的误判。

7.属性的遍历（枚举：列出某些有穷序列集的所有成员的程序，或者是一种特定类型对象的计数）

for…in循环遍历对象自身的和继承的可枚举属性（不含 Symbol 属性）

Object.keys返回一个数组，包括对象自身的（不含继承的）所有可枚举属性（不含 Symbol 属性）的键名

Object.getOwnPropertyNames返回一个数组，包含对象自身的所有属性（不含 Symbol 属性，但是包括不可枚举属性）的键名

8.Symbol：可以保证不会与其他属性名产生冲突，不能与其他类型的值进行运算

Symbol.match
Symbol.replace
Symbol.search
Symbol.split

9.Set可以去除字符串里面的重复字符
Set结构的实例属性：

Set.prototype.constructor：构造函数，默认就是Set函数。
Set.prototype.size：返回Set实例的成员总数。

Set实例的方法

1.操作方法（操作数据)
Set.prototype.add(value)：添加某个值，返回 Set 结构本身。
Set.prototype.delete(value)：删除某个值，返回一个布尔值，表示删除是否成功。
Set.prototype.has(value)：返回一个布尔值，表示该值是否为Set的成员。
Set.prototype.clear()：清除所有成员，没有返回值。

2.遍历方法（遍历成员）
Set.prototype.keys()：返回键名的遍历器
Set.prototype.values()：返回键值的遍历器
Set.prototype.entries()：返回键值对的遍历器
Set.prototype.forEach()：使用回调函数遍历每个成员

Array.from方法可以将 Set 结构转为数组。

10.Map结构的属性和操作方法

size//返回Map结构的成员总数
Map.prototype.set(key,value)//生成键，设置键名和键值，然后返回Map结构
Map.prototype.get(key)//读取key对应的键值
Map.prototype.has(key)//返回布尔值，表示某个键是否在当前Map对象中
Map.prototype.delete(key)//删除成功返回true
Map.prototype.clear()//清除所有成员，没有返回值

遍历方法（Map的遍历顺序就是插入顺序）

Map.prototype.keys()：返回键名的遍历器
Map.prototype.values()：返回键值的遍历器
Map.prototype.entries()：返回所有成员的遍历器
Map.prototype.forEach()：遍历Map的所有成员

11.Promise是一个对象，也是一个构造函数

Promise 对象后面要跟catch方法,可以处理Promise内部发生的错误。catch方法返回的还是一个 Promise 对象，后面还可以接着调用then方法。

为异步操作提供统一接口

Promise对象通过自身的状态，来控制异步操作：
1.异步未完成（pending）
2.异步成功（fulfilled）,传回value，状态变为fulfilled
3.异步失败（rejected)，抛出error，状态变为rejected

Promise实例的then方法，用来添加回调函数

12.async:返回值是Promise

async函数返回一个 Promise 对象，可以使用then方法添加回调函数。当函数执行的时候，一旦遇到await就会先返回，等到异步操作完成，再接着执行函数体内后面的语句。

await命令后面的 Promise 对象如果变为reject状态，则reject的参数会被catch方法的回调函数接收到。
任何一个await语句后面的 Promise 对象变为reject状态，那么整个async函数都会中断执行。
如果await后面的异步操作出错，那么等同于async函数返回的 Promise 对象被reject。
