实习总结

2019-08-30 阅读次数：
目录
回到顶部

多积累多思考！
理论要结合实践！

两个月的前端实习到今天就结束了（其实只有7周），回顾一下这两个月大概可以分为3个阶段吧。

前两个星期基本上都是在看各种文档，像 git 的常用命令（其实在学校的时候也有了解过一下下 git，不过基本没怎么用过命令，还有svn啥的）、Es6 入门、一些包管理工具（yarn 和 npm），然后才是 react 的文档。

弱弱地说一下，其实这些文档看了总觉得好像也不是很难，后来慢慢开始结合实际代码之后，发现真的是一头雾水，完全没有真正地理解下来。

第三周开始配置示例，然后提了 mr，应该才是真正地开始用到 git 的常用命令。抓住第三周的尾巴开始做小小的需求（固定 sidebar，让 svg 自适应），一做就发现基础真的很不扎实。

在做这个需求的过程中，开始用到之前看过的 react 方面的生命周期（16.3前）、状态提升（最后把它理解成父子组件间的通信，这个东西贯穿到下一个需求）、还有关于布局的等等等等，终于慢慢开始理解。
https://user-gold-cdn.xitu.io/2017/11/11/88e11709488aeea3f9c6595ee4083bf3?imageslim

然后还需要进行单元测试（这也是很重要的一步啊），用到 Jest 测试框架和 Enzyme 测试库。

7月就这样很快过去，然后开始了做了整个8月的需求。

进一步理解了父子组件间的通信、新的生命周期（16.4后）（最后也没有用到）、用到 yarn 导入依赖。

https://user-gold-cdn.xitu.io/2018/8/12/1652a030ed1506e0?imageslim

这一整个月里，似乎 get 到了一点点导师的思考方式，有点棒哈哈哈。
